Memorizador de Textos

Escolha um JSON, selecione uma fase e monte cada trecho clicando nas palavras.

Você pode selecionar um ou mais arquivos de livro .json. O progresso pode ser salvo e restaurado por outro JSON.

Fases

Cada parágrafo vira uma fase. As fases com mais erros aparecem destacadas.

Fase 0 de 0 • Trecho 0 de 0 • Erros neste trecho: 0

A palavra errada fica vermelha até você acertar a próxima. Ela só conta erro uma vez nesse ciclo.

Nenhum texto carregado ainda.

Este app aceita os dois formatos abaixo:

[
  {
    "titulo": "Meu texto",
    "paragrafos": [
      "Primeiro parágrafo.",
      "Segundo parágrafo."
    ]
  }
]
{
  "titulo": "Meu texto",
  "paragrafos": [
    {
      "id": "p001",
      "texto": "Primeiro parágrafo completo.",
      "frases": [
        { "texto": "Primeira frase." },
        { "texto": "Segunda frase." }
      ]
    }
  ]
}

Texto revelado da fase

Complete o primeiro trecho para começar a revelar o parágrafo.
